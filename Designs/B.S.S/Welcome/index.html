PrograMeet

PrograMeet was created because we realized that there was a large disconnect between the Code Next community. A majority of the people in the Chelsea, Harlem and California cohorts don't know each other. Along with this, most people in the same cohort don't even know each other. To fix this problem, we created PrograMeet, a platform that connects members of the Code Next community and encourages collaboration.

Network

Connect with other Code Next students, coaches and Googlers. Meet people in our community and form connections with other members of our community.

Track attendance

Code Next coaches are able to track attendance from all events.

Collaborate

Meet other Code Next students, coaches, Googlers and ask for help with projects, work on projects with others, post questions and get them answered, look for friends, business partners, host/suggest events, set up meetings with Googlers and coaches and more!
