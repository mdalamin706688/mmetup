PrograMeet

PrograMeet was created because of the disconnect between the Code Next family. Most of the people in the Chelsea, Harlem and Oakland cohorts don't know each other and most people in the same cohort don't know each other. To fix this problem, we created PrograMeet, the platform that connects members of the Code Next family and encourages collaboration.




Network

Connect with other Code Next students, coaches and Googlers. Meet people in our community and form connections with other members of our community.

Track attendance

Code Next coaches are able to track attendance from all events.

Collaborate

Meet other Code Next students, coaches, Googlers and ask for help with projects, work on projects with others, post questions and get them answered, look for friends, business partners, host/suggest events, set up meetings with Googlers and coaches and more!

Book time

Quickly schedule time to meet with coaches and Googlers to speak with them.

Calendar

View and join upcoming events in the Code Next community such as clubs, bootcamps, summer programs and many more!

Resources

Gain quicker access to important Code Next forms along with important education resources. Learn about free online programs led by America's top universities that teach topics such as software engineering, accounting, history, English literature, macroeconomics and many more!
